# Naja 智能交易系统 - 营销介绍

## 一句话介绍

**Naja** 是一款基于「注意力机制+认知科学+佛教八识理论」打造的自主进化交易决策系统，让机器像经验丰富的交易员一样，感知市场、理解叙事、觉醒交易直觉。

---

## 核心亮点

### 1. 像真人大脑一样思考

传统量化系统只会「执行规则」，Naja 具备：

- **感知能力**：波动率曲面、预尝信号，行情变化提前预判
- **理解能力**：追踪市场叙事，感知恐慌与贪婪情绪
- **决策能力**：矛盾检测、仓位自适应、风险实时管控
- **进化能力**：从交易结果中学习，自动生成和优化策略

### 2. 八识觉醒架构

| 八识 | Naja 实现 | 功能 |
|------|----------|------|
| 前五识 | ProphetSense + VolatilitySurface | 预感知、波动率 |
| 第六识 | FirstPrinciplesMind | 因果追踪、矛盾检测 |
| 第七识 | AdaptiveManas + MetaEvolution | 顺应决策、策略进化 |
| 第八识 | AwakenedAlaya | 模式召回、顿悟机制 |

### 3. 双向协同 · 知行合一

```
认知系统（理解市场）
    ↑
    │ 叙事 / 情绪 / 洞察
    ↓
觉醒系统（做出决策）
    ↑
    │ 交易执行 / 结果反馈
    ↓
认知系统（学习进化）
```

系统不仅能「知」，更能「行」，并且从行动结果中不断进化。
