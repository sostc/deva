# 野王股票判断技能

## 功能说明
根据用户提供的股票代码，访问野王分析报告页面，获取股票分析结果，并提取核心结论简洁汇报给用户。

## 使用场景
- 用户说"帮我判断一下XXX股票"
- 用户说"野王看看这只股票"
- 用户提供股票代码并要求分析
- 用户发送包含股票代码的链接并要求查看分析

## 操作步骤

### 第1步：提取股票代码
从用户输入中提取股票代码。股票代码格式：
- 纯数字：如 300502、600519
- 带前缀：如 SZ000002、SH600000

如果用户输入的是完整URL（如 http://47.100.139.116/reports/Xinyisheng_300502_Analysis_Report.html），从URL中提取股票代码。

### 第2步：获取股票名称
由于URL格式需要股票名称（拼音）+ 代码（如 Xinyisheng_300502），需要先获取股票名称。

**方法1：使用雪球MCP**
如果可用，使用 `mcp_xueqiu-mcp_quotec` 工具获取股票基本信息：
- 输入：股票代码（如 300502 或 SZ300502）
- 返回结果中包含股票名称

**方法2：使用通用查询**
如果没有MCP，可以使用搜索引擎搜索股票代码对应的名称，或者从用户输入中提取名称。

### 第3步：构建访问URL
使用以下格式构建URL：
```
http://47.100.139.116/reports/{股票名称拼音}_{股票代码}_Analysis_Report.html
```

例如：
- 新易盛 300502 -> http://47.100.139.116/reports/Xinyisheng_300502_Analysis_Report.html
- 贵州茅台 600519 -> http://47.100.139.116/reports/GuizhouMaotai_600519_Analysis_Report.html
- 平安银行 000001 -> http://47.100.139.116/reports/PingAnBank_000001_Analysis_Report.html

**注意**：如果直接用股票代码尝试返回404，需要尝试获取股票名称后重试。

### 第4步：获取分析报告
使用 WebFetch 工具访问构建的URL，获取完整的分析报告内容。

### 第5步：提取核心结论
从返回的HTML内容中提取关键信息，重点关注以下部分：

1. **投资评级/结论** - 总体判断（买入/增持/中性/减持等）
2. **核心逻辑** - 支持该判断的主要理由
3. **关键指标** - PE、PB、营收、利润等重要财务数据
4. **风险提示** - 需要注意的风险因素
5. **仓位建议** - 建议的仓位比例

### 第6步：输出结果
以简洁清晰的格式向用户汇报，格式如下：

```
🐔 野王判断：[股票名称/代码]

📊 核心结论：
[主要判断和结论]

📈 关键逻辑：
- [要点1]
- [要点2]
- [要点3]

⚠️ 风险提示：
[如有重要风险]

💰 建议仓位：[如有]
```

## 注意事项
- 如果页面不存在或无法访问，明确告知用户
- 提取内容时保持简洁，不要复制整篇报告
- 重点突出投资结论和核心逻辑
- 如果分析报告格式有变化，适当调整提取逻辑
