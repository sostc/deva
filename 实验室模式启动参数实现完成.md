## ✅ NAJA 启动模式说明

### 三种启动模式

| 模式 | 命令行参数 | 说明 |
|------|-----------|------|
| **普通实盘模式** | (无特殊参数) | 根据交易时间自动运行 |
| **强制实盘调试模式** | `--force-realtime` | 忽略交易时间，强制获取实盘数据 |
| **实验室模式** | `--lab` | 回放历史数据进行测试 |

---

### 1. 普通实盘模式（默认）

```bash
python -m deva.naja
```

- 根据交易时间自动开始/停止数据获取
- 交易时间：周一至周五 9:30-11:30, 13:00-15:00

---

### 2. 强制实盘调试模式

```bash
# 启用强制实盘调试（忽略交易时间）
python -m deva.naja --force-realtime

# 可选：指定日志级别
python -m deva.naja --force-realtime --log-level DEBUG
```

**用途：**
- 非交易时间调试实盘数据获取
- 周末测试系统功能
- 强制获取 Sina 实时行情

**UI 显示：**
- 📡 实盘获取器 🟢 强制调试中
- 🔧 模式: 强制实盘(忽略交易时间)

---

### 3. 实验室模式（回放测试）

```bash
# 启动实验室模式，使用指定的历史行情回放表
python -m deva.naja --lab --lab-table quant_snapshot_5min_window

# 自定义回放间隔（秒）
python -m deva.naja --lab --lab-table quant_snapshot_5min_window --lab-interval 0.5

# 自定义回放速度倍数
python -m deva.naja --lab --lab-table quant_snapshot_5min_window --lab-speed 2.0

# 仅启用注意力系统，不回放数据
python -m deva.naja --lab
```

**参数说明：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--lab` | 启用实验室模式 | 关闭 |
| `--lab-table` | 回放数据表名 | 无 |
| `--lab-interval` | 回放间隔（秒） | 1.0 |
| `--lab-speed` | 回放速度倍数 | 1.0 |
| `--lab-debug` | 输出详细处理信息 | 关闭 |

---

### 4. 其他启动参数

```bash
# 新闻雷达模式
python -m deva.naja --news-radar-speed 10    # 10倍速新闻雷达
python -m deva.naja --news-radar-sim         # 新闻雷达模拟模式

# 认知系统调试
python -m deva.naja --cognition-debug        # 完整认知调试模式

# 调参模式
python -m deva.naja --tune --lab-table quant_snapshot_5min_window
python -m deva.naja --tune --tune-method random --tune-samples 50
```
