## ✅ 实验室模式启动参数实现完成

### 使用方法

```bash
# 启动实验室模式，使用指定的历史行情回放表
python -m deva.naja --lab --lab-table quant_snapshot_5min_window

# 自定义回放间隔（秒）
python -m deva.naja --lab --lab-table quant_snapshot_5min_window --lab-interval 0.5

# 自定义回放速度倍数
python -m deva.naja --lab --lab-table quant_snapshot_5min_window --lab-speed 2.0

# 仅启用注意力系统，不回放数据
python -m deva.naja --lab
```

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--lab` | 启用实验室模式 | 关闭 |
| `--lab-table` | 回放数据表名 | 无 |
| `--lab-interval` | 回放间隔（秒） | 1.0 |
| `--lab-speed` | 回放速度倍数 | 1.0 |
