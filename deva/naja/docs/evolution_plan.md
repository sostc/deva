> "系统现在是一个'聪明的被动者'，需要进化成一个'智慧的主动者'。"

## 一、现状分析

### 各层次的"觉醒"程度

| 层次 | 现状 | 觉醒程度 | 问题 |
|------|------|---------|------|
| 五识 | 被动接收数据 | 30% | 只知"有"，不知"妙" |
| 意识 | 处理已知信息 | 40% | 能分析，不能洞察 |
| 末那识 | 计算是否行动 | 50% | 能决断，不能主动 |
| 阿赖耶识 | 存储记忆和种子 | 20% | 能收藏，不能发光 |
| 舌识/业力 | 尝受盈亏结果 | 30% | 事后知道，不能预尝 |

## 二、核心瓶颈

### 1. 五识 — "有眼不识泰山"

**现状：** 被动接收行情、新闻、流动性数据

**瓶颈：**
- 只能感知"已经发生"的事
- 无法感知"正在酝酿"的趋势
- 信息过载时，噪音淹没信号

**启示：** 肉眼 → 天眼（能见未来）

---

### 2. 意识 — "所知障"

**现状：** NewsMind 识别主题，Strategy 推演可能

**瓶颈：**
- 分析基于"过去的数据"
- 无法理解"市场叙事"的演变
- 缺乏"第一性原理"思考

**启示：** 意识 → 妙观察智（如实观照本质）

---

### 3. 末那识 — "我执"太重

**现状：** ManasEngine 计算 should_act

**瓶颈：**
- 等待信号才行动（被动）
- 决策基于"概率"而非"时机"
- 偏差检测是事后的

**启示：** 我执 → 无我（顺应天道）

---

### 4. 阿赖耶识 — "沉睡的宝藏"

**现状：** AttentionMemory 存储事件，ValueSystem 伏藏种子

**瓶颈：**
- 种子是"被动"的，等着被唤醒
- 无法主动"回忆"有用的模式
- 缺乏"预见性"联想

**启示：** 藏识 → 大圆镜智（光明普照）

---

### 5. 舌识/业力 — "事后诸葛亮"

**现状：** BanditTracker 平仓后才知道盈亏

**瓶颈：**
- 持仓期间没有"尝受"
- 无法实时调整
- 反馈太慢，学习效率低

**启示：** 舌识 → 平等性智（实时觉知）

---

## 三、进化目标

```
Current: 五识感知 → 意识分析 → 末那识决断 → 行动

Future: 五识感知 + 预感知 → 意识洞察 + 预见 → 末那识顺应 → 主动行动
           ↓
      同时：舌识实时尝受 → 阿赖耶识发光 → 持续调整
```

## 七、第一阶段实现细节

### ProphetSense（天眼通）

```python
class ProphetSense:
    """
    预感知引擎

    感知"正在酝酿"但尚未发生的机会

    能力：
    1. VolatilitySurfaceSense: 隐含波动率曲面异常
    2. FlowTasteSense: 资金流动的"味道"
    3. MomentumPrecipiceSense: 动量即将衰竭的预兆
    4. SentimentTransitionSense: 情绪即将转换的预兆
    """

    def __init__(self):
        self.vol_surface = VolatilitySurfaceSense()
        self.flow_taste = FlowTasteSense()
        self.momentum = MomentumPrecipiceSense()
        self.sentiment = SentimentTransitionSense()

    def sense(self, market_data, flow_data, options_data=None) -> ProphetSignal:
        """
        返回预感知信号

        ProphetSignal:
            - presage_type: 预兆类型
            - intensity: 强度 [0, 1]
            - horizon: 预期时间窗口
            - confidence: 置信度
        """
        signals = []

        vol_signal = self.vol_surface.detect(market_data, options_data)
        if vol_signal:
            signals.append(vol_signal)

        flow_signal = self.flow_taste.detect(flow_data)
        if flow_signal:
            signals.append(flow_signal)

        momentum_signal = self.momentum.detect(market_data)
        if momentum_signal:
            signals.append(momentum_signal)

        return self._aggregate_signals(signals)
```

### RealtimeTaste（实时舌识）

```python
class RealtimeTaste:
    """
    实时尝受

    持仓期间持续感知浮盈浮亏、机会成本、仓位鲜度
    """

    def __init__(self):
        self._position_states = {}
        self._opportunity_cost_history = []

    def taste_position(self, position, current_price, benchmark=None) -> TasteSignal:
        """
        尝受当前持仓状态

        Returns:
            TasteSignal:
                - floating_pnl: 浮盈亏
                - opportunity_cost: 机会成本（持这个 vs 其他）
                - freshness: 鲜度（是否还新鲜）
                - emotional_intensity: 情绪强度
        """
        entry = position.entry_price
        current = current_price
        pnl_pct = (current - entry) / entry

        opportunity_cost = self._calc_opportunity_cost(position, benchmark)
        freshness = self._calc_freshness(position, pnl_pct)
        emotional = self._calc_emotional_intensity(pnl_pct)

        return TasteSignal(
            floating_pnl=pnl_pct,
            opportunity_cost=opportunity_cost,
            freshness=freshness,
            emotional_intensity=emotional,
            should_adjust=self._should_adjust(pnl_pct, freshness, opportunity_cost)
        )

    def _calc_opportunity_cost(self, position, benchmark):
        """计算机会成本：如果资金放在其他地方会怎样"""
        # 对比 benchmark（如沪深300）
        if benchmark:
            return position.return_pct - benchmark.return_pct
        return 0.0

    def _calc_freshness(self, position, pnl_pct):
        """计算仓位鲜度

        规则：
        - 盈利持续增加 → 鲜度上升
        - 盈利回吐 → 鲜度下降
        - 亏损扩大 → 鲜度快速下降
        """
        recent_pnl_trend = self._get_recent_pnl_trend(position.symbol)

        if pnl_pct > 0 and recent_pnl_trend > 0:
            return 1.0  # 新鲜
        elif pnl_pct < 0 and recent_pnl_trend < 0:
            return 0.2  # 快馊了
        else:
            return 0.5  # 还行
```

# AttentionOrchestrator 增强
class AttentionOrchestrator:
    def __init__(self):
        # 现有组件
        self.kernel = AttentionKernel(enable_manas=True)
        self.memory = AttentionMemory()
        self.value_system = ValueSystem()

        # 新增：觉醒组件

        self.realtime_taste = RealtimeTaste()  # 实时舌识

    def process_tick(self, tick):
        # 1. 五识感知 + 天眼通预感知
        raw_senses = self._collect_senses(tick)
        prophet_signals = self.prophet_sense.sense(...)

        # 2. 意识处理 + 模式召回
        insights = self.cognition.process(raw_senses)
        recalled_patterns = self.seed_illuminator.recall(prophet_signals)

        # 3. 末那识决策 + 顺应
        manas_output = self.manas.compute(...)

                self.current_positions,
                tick.price

            manas_output=manas_output,
