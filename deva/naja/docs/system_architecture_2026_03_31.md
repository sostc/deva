## 一、系统定位

Naja 是一个**自主进化的交易决策系统**，融合了注意力机制、认知科学和佛教"八识"理论。

**核心目标**：让机器能够像经验丰富的交易员一样，"感知"市场、"理解"叙事、"觉醒"出交易直觉。

## 三、各系统职责

## 五、认知 vs 觉醒

| 对比维度 | 认知系统 | 觉醒系统 |
|---------|---------|---------|
| **定位** | 军师（分析局势） | 将军（做出决策） |
| **回答** | "市场发生了什么？为什么？" | "现在买还是卖？买多少？" |
| **数据来源** | 新闻、舆情、研报 | 市场数据、交易信号 |
| **输出** | 叙事、情绪、洞察 | 信号、仓位、置信度 |
| **记忆** | InsightPool（知识积累） | AwakenedAlaya（模式召回） |

**关系**：认知系统服务"知"，觉醒系统服务"行"，两者互补实现"知行合一"。

## 八、注意力调度详解

**Center.py** 是真正的"调度者"，协调所有系统干活：

```python
on_market_data(data):
    # 1. Pipeline处理数据
    data = self._pipeline.process(data)

    # 2. 更新注意力
    self._update_attention(data)

    # 3. 通知认知系统
    self._notify_cognition()

    # 4. 处理觉醒模块
    self._process_awakened_modules(data, symbol_market_data)

    # 5. 策略分发
    signals = strategy_mgr.process_data(data, context)

    # 6. 应用觉醒调整（波动率降仓、矛盾检测）
    signals = self._apply_awakening_adjustments(signals, data)

    # 7. 应用认知反馈（情绪、热点）
    signals = self._apply_cognition_to_signals(signals, data)

    # 8. 预尝过滤
    signals = self._apply_pre_taste_filter(signals, data)

    # 9. 执行交易
    self._execute_signals(signals)

    # 10. 反馈结果到觉醒系统
    self._feedback_to_awakened_systems(signal, result)
```
