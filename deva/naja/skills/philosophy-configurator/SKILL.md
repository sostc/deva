# Philosophy Configurator - 理念配置生成器

## 概述

将用户的投资哲学（天道民心、价值观、愿景使命）自动转化为 Naja 智慧系统的完整配置体系。

**核心能力：**
- 分析用户理念描述，提取核心概念
- 生成价值观类型配置
- 生成天道/民心关键词体系
- **自动拆解所选行业的供应链**
- **根据供应链扩展天道关键词**
- 生成系统配置（SOUL.md、Values、Profile等）
- 一键应用到系统中
- **支持多套灵魂方案管理**（创建/切换/删除/导出）

## 灵魂管理系统

系统支持**多套灵魂方案**，每套灵魂包含完整的配置体系：

## 供应链配置详解

### 内置行业供应链模板

| 行业 | 供应链层级 | 核心节点 | 天道信号来源 |
|------|-----------|---------|------------|
| **AI** | 上游 | GPU(HBM)、光刻机、先进封装 | 算力告急、产能不足 |
| **AI** | 中游 | 大模型(训练/推理) | 技术突破、效率提升 |
| **AI** | 下游 | 应用层(Agent、AIGC) | 商业化、用户增长 |
| **芯片** | 上游 | 硅片，光刻胶、特气 | 产能、良品率 |
| **芯片** | 中游 | 晶圆制造、封装测试 | 制程突破、良率提升 |
| **芯片** | 下游 | IC设计、产品应用 | 需求爆发、国产替代 |
| **新能源** | 上游 | 锂矿、钴矿、硅料 | 资源价格、产能 |
| **新能源** | 中游 | 电池片、组件 | 技术路线、成本 |
| **新能源** | 下游 | 整车、储能 | 渗透率、订单 |

### 供应链关键词扩展

根据用户选择的行业，系统自动：
1. 提取供应链各节点的关键词
2. 扩展到天道关键词库
3. 生成供应链节点映射

## 脚本接口

### 快速使用

```python
from deva.naja.config.soul_generator import SoulGenerator

generator = SoulGenerator()

soul = generator.generate(
    name="🚀 我的灵魂",
    direction="creative",  # creative 或 speculative
    domains=["AI", "芯片", "新能源"],
    risk_level="moderate",  # conservative, moderate, aggressive
    custom_notes="重点关注算力短缺问题"
)

result = generator.apply(soul)
print(result)
```

### SoulManager 多灵魂操作

```python
from deva.naja.config.soul_manager import SoulManager

manager = SoulManager()

# 列出所有灵魂
souls = manager.list_souls()

# 创建新灵魂
manager.create_soul(
    name="🚀 激进型",
    philosophy={...}
)

# 激活灵魂
manager.activate_soul("激进型")

# 导出灵魂
manager.export_soul("激进型", "/path/to/soul.json")

# 导入灵魂
manager.import_soul("/path/to/soul.json")

# 切换（有持仓检测）
manager.switch_soul("保守型")
```

1. **保留现有配置**：如果用户不特别要求，不覆盖现有工作配置
2. **渐进式更新**：可以只更新部分配置（SOUL/Values/Narrative）
3. **配置验证**：应用后必须验证系统各模块能正确读取
4. **回滚支持**：建议在重大更新前备份现有配置
5. **数据库持久化**：配置保存到文件实现跨会话持久化
6. **多灵魂隔离**：每个灵魂的配置完全隔离，互不影响
7. **持仓保护**：已有持仓按原灵魂管理，新仓位用新灵魂
8. **激活生效**：只有激活的灵魂配置会应用到系统
